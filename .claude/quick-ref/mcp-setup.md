# MCP Servers - Quick Reference

> **Purpose**: AI-assisted dev tools  
> **Verification:** docs/mcp-verification-checklist.md

Project supports 6 MCP servers:

1. Playwright - E2E automation.
2. Storybook - Component docs (build first).
3. shadcn UI - UI library docs.
4. Vite - Auto with dev server.
5. Context7 - Library lookup.
6. Graphiti Memory - Knowledge persistence.

**Usage:** Query shadcn/Context7 before coding; build Storybook for MCP.

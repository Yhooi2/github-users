# GitHub Analytics Design System

Welcome to the **GitHub Analytics Design System** - a comprehensive glassmorphism-based UI kit built with React 19, TypeScript, and Tailwind CSS v4.

## Overview

This design system provides a complete set of components for building beautiful, modern GitHub analytics interfaces. All components support three themes:

- **Light** - Clean, bright appearance for daytime use
- **Aurora** - Dark theme with aurora borealis inspired colors
- **Glass** - Full glassmorphism with blur effects and floating orbs

## Component Categories

### Core Glass Components
The foundation of our design system:
- `GlassCard` - Container with blur effect and optional glow
- `GlassButton` - Buttons with primary, secondary, and ghost variants
- `GlassProgress` - Progress bars with gradient fills
- `GlassBadge` - Status badges with semantic colors
- `GlassInput` - Text inputs with glass styling
- `GlassToggle` - Toggle switches
- `GlassSelect` - Dropdown selects

### Status & Metrics
Components for displaying data:
- `StatusIndicator` - Traffic light status (green/yellow/red)
- `MetricCard` - Metric display with color variants
- `LanguageBar` - Language distribution visualization

### Display Components
- `Avatar` - User avatars with initials and gradients
- `Alert` - Alert messages with variants

### Desktop Components
Specialized components for desktop layouts:
- `SearchBar` - Search input with button
- `TabToggle` - Tab switching component
- `YearCard` - Year statistics card
- `RepoCard` - Repository card with expandable details
- `AIInsightsCard` - AI summary CTA
- `FlagAlert` - Flag/issue alerts
- `IconButton` - Icon-only buttons

### Layout Components
- `Background` - Full-screen background with orbs and grid
- `Header` - Sticky header with blur effect
- `ThemeToggle` - Theme switcher button

## Design Tokens

Our design system uses a comprehensive token system with 170+ tokens per theme covering:
- Background gradients and orbs
- Glass effects (subtle/medium/strong)
- Text colors (primary/secondary/muted/accent)
- Status colors (green/yellow/red)
- Metric colors (emerald/amber/blue/red)
- Button and badge variants

## Getting Started

```tsx
import {
  ThemeProvider,
  Background,
  GlassCard,
  GlassButton,
} from './design_system';

function App() {
  return (
    <ThemeProvider defaultTheme="glass">
      <Background>
        <GlassCard intensity="strong" glow="violet">
          <h1>Hello World</h1>
          <GlassButton variant="primary">
            Get Started
          </GlassButton>
        </GlassCard>
      </Background>
    </ThemeProvider>
  );
}
```

## Theme Context

Access theme information anywhere in your app:

```tsx
import { useTheme, useThemeStyles } from './design_system';

function MyComponent() {
  const { theme, cycleTheme } = useTheme();
  const t = useThemeStyles();

  return (
    <div style={{ color: t.textPrimary }}>
      Current theme: {theme}
    </div>
  );
}
```

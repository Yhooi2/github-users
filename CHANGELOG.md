### BREAKING CHANGES

- **useUserAnalytics:** Promise.all replaced with try-catch per query

**Problem**:

- Promise.all() rejects if ANY year query fails
- Entire timeline becomes empty even with partial failures
- parallel-query-failure integration tests all failing (0/4)

**Solution**:

- Wrap each year query in try-catch block
- Failed queries return null instead of throwing
- Filter out null values to keep only successful years
- console.warn logs individual failures for debugging

- Import generateYearRanges() for exact date matching
- Use yearRanges[i].from/to instead of manual date construction
- Fix mockUserProfile.createdAt to match year range mocks
